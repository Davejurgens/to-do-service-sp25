# Code examples

**Author:** Dave Jurgens

## cURL example
This cURL request pulls Marvel comic book information.

### cURL command

```shell
curl http://localhost:3000/marvelComics
```

### cURL response

```shell
{
    "superHero": "Captain_America",
    "villain": "Doctor_Doom",
    "superPet": "Lockjaw",
    "civilians": "Mary_Jane",
    "total": "4"
  }
]
```
